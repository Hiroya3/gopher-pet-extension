# Gopher Pet

A VS Code / Cursor extension that adds an animated Gopher pet to your Explorer sidebar. Watch it run back and forth, or play a mini jump game to avoid obstacles.

## Features

- **Animated Gopher**: A cute Gopher runs continuously in the Explorer sidebar panel
- **Jump Game**: Start a game where you jump over stones to score points
- **Game Over Effects**: Collision triggers spinning animation and screen shake

## Usage

Once installed, the Gopher Pet panel appears in the Explorer sidebar. The gopher will automatically start running.

### Commands

- **Gopher: Start Game** - Activates game mode from the command palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)

### Game Controls

| Key | Action |
|-----|--------|
| Space | Jump (during game) / Restart (after game over) |
| Esc | Exit game and return to idle animation |

## Requirements

- VS Code 1.105.0 or higher
- Cursor (compatible)
